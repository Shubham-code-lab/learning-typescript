Typescript is javascript superset.
Typescript can't be executed in javascript environment like browser (node.js also can't typescript)
It tool a compiler to compile a .ts in .js
Extra feature provide by ts are compiled in js with workaround and also it run before runtime compilation so it throw error earlier

Overview:-
1]Type
2]next-gen javascript code like bevel compile ts to older js
3]non-js feature like interface or generic type
4]meta-programming feature like decorator
5]rich configure option(make it more strict or less)
